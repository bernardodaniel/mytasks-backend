# MyTasks

### TarefaService:
* métodos para
* listar todas tarefas
* salvar uma tarefa
* atualizar uma tarefa

### TarefaController:
* GET - todas tarefas
* GET - uma tarefa por id
* POST - salvar tarefa
* PUT - atualizar tarefa

### DTO's
* TarefaRequest
* TarefaResponse
* Utilizar ModelMapper

### Validação dos dados 
* BeanValidation
* Criar CustomGlobalExceptionConfiguration
